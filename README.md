# 🏋️ Obesity Level Prediction using K-Nearest Neighbors (KNN)

## 📌 Project Overview

This project focuses on predicting **obesity levels** using the **K-Nearest Neighbors (KNN)** classification algorithm.  
The dataset contains information about individuals' lifestyle habits, physical conditions, and eating behaviors. Using these features, the model predicts the obesity category of a person.

This project demonstrates how Machine Learning can be used in **health analytics** to identify potential obesity risks based on lifestyle data.

---

## 🎯 Objectives

- Perform Exploratory Data Analysis (EDA) on obesity dataset
- Clean and preprocess lifestyle and health-related data
- Encode categorical variables
- Apply feature scaling (important for KNN)
- Train and evaluate a KNN classification model
- Predict obesity levels based on individual attributes

---

---

## 🔹 Project Workflow

1. Load and explore the obesity dataset
2. Perform Exploratory Data Analysis (EDA)
3. Handle missing values and inconsistent data
4. Encode categorical variables
5. Apply feature scaling using StandardScaler
6. Split dataset into training and testing sets
7. Train the **KNN classification model**
8. Evaluate model performance using:
   - Accuracy Score
   - Confusion Matrix
   - Classification Report
9. Predict obesity category for new data

---

## 📊 Key Insights

- Lifestyle factors such as **physical activity, food habits, and transportation methods** influence obesity levels.
- Proper feature scaling significantly improves **KNN performance**.
- KNN works effectively for **health-related classification problems**.
